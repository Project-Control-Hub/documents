> 반응형/접근성 기준은 [반응형/접근성 가이드](../반응형_접근성_가이드.md)를 참조한다.

## 0. 공통 UI 요소

### GNB (Global Navigation Bar)

모든 화면(SCR-001 제외)에 적용되는 상단 글로벌 내비게이션 바.

| 요소 ID | 유형 | 레이블 | 비고 |
|---------|------|--------|------|
| gnb-logo | Image/Link | 서비스 로고 | 클릭 시 SCR-002 메인 대시보드로 이동 |
| gnb-project-select | Select | 프로젝트 선택 | 접근 가능한 프로젝트 목록 드롭다운 |
| gnb-nav-backlog | Link | 백로그 | SCR-003으로 이동 |
| gnb-nav-board | Link | 보드 | SCR-004(스크럼) 또는 SCR-005(칸반)으로 이동 |
| gnb-nav-reports | Link | 리포트 | SCR-012로 이동 |
| gnb-search | Input | 검색 | 클릭 시 SCR-008 JQL 검색 이동, 단축키: `/` |
| gnb-create | Button | 만들기 | 이슈 생성 모달 또는 SCR-007로 이동 |
| gnb-notifications | IconButton | 알림 | 알림 목록 드롭다운, 미확인 알림 수 뱃지 표시 |
| gnb-user-menu | Avatar/Menu | 사용자 메뉴 | 프로필 / 설정 / 로그아웃 |
| gnb-settings | IconButton | 프로젝트 설정 | SCR-013으로 이동 (ADMIN 권한 시 표시) |

#### GNB 기능 정의

| 기능 ID | 기능명 | 트리거 | 동작 | API |
|---------|--------|--------|------|-----|
| FN-GNB-01 | 프로젝트 전환 | gnb-project-select 변경 | 선택한 프로젝트의 기본 화면(백로그)으로 이동 | GET /project |
| FN-GNB-02 | 알림 조회 | gnb-notifications 클릭 | 최근 알림 목록 드롭다운 표시, 클릭 시 해당 이슈로 이동 | GET /notifications |
| FN-GNB-03 | 빠른 이슈 생성 | gnb-create 클릭 | 이슈 생성 폼 모달 표시 (현재 프로젝트 자동 선택) | POST /rest/api/3/issue |
| FN-GNB-04 | 로그아웃 | gnb-user-menu → 로그아웃 | 토큰 무효화 후 SCR-001 로그인으로 이동 | POST /auth/logout |

## 2. 화면 상세 정의

### SCR-001: 로그인

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-001 |
| 화면명 | 로그인 |
| URL | /login |
| 설명 | 사용자 인증을 위한 로그인 화면. JWT 토큰 발급 및 자동 갱신(Token Refresh) 지원 |

#### UI 요소

| 요소 ID | 유형 | 레이블 | 필수 | 유효성 검사 | 비고 |
|---------|------|--------|------|-------------|------|
| inp-email | Input | 이메일 | Y | 이메일 형식 | placeholder: "이메일을 입력하세요" |
| inp-password | Input | 비밀번호 | Y | 최소 8자 | type: password |
| btn-login | Button | 로그인 | - | - | Primary |
| lnk-forgot-password | Link | 비밀번호를 잊으셨나요? | - | - | /forgot-password 이동 |
| chk-remember | Checkbox | 로그인 상태 유지 | - | - | Refresh Token 만료 기간 연장 |

#### 기능 정의

| 기능 ID | 기능명 | 트리거 | 동작 | API |
|---------|--------|--------|------|-----|
| FN-001-01 | 로그인 | btn-login 클릭 | 이메일/비밀번호 검증 후 API 호출, Access Token(1h) 및 Refresh Token(7d) 로컬 저장 | POST /auth/login |
| FN-001-02 | 유효성 검사 | 입력 시 | 실시간 입력값 검증 (이메일 형식, 비밀번호 8자 이상) | - |
| FN-001-03 | 계정 잠금 | 5회 실패 | 30분간 로그인 차단, 잠금 해제 시각 표시 | - |
| FN-001-04 | Token Refresh | Access Token 만료 시 | 백그라운드에서 Refresh Token으로 Access Token 자동 재발급. Refresh Token도 만료 시 로그인 화면 강제 이동 | POST /auth/refresh |
| FN-001-05 | 비밀번호 찾기 | lnk-forgot-password 클릭 | 이메일 입력 → 재설정 링크 발송 안내 화면 이동 | POST /auth/forgot-password |

#### 상태 처리

| 상태 | 표시 내용 | 조건 |
|------|-----------|------|
| 로딩 | 스피너 표시, btn-login 비활성화 | API 호출 중 |
| 성공 | 대시보드 이동 | 로그인 성공 |
| 실패 | "이메일 또는 비밀번호가 올바르지 않습니다" 인라인 에러 | 인증 실패 |
| 잠금 | "계정이 잠겼습니다. 30분 후 재시도하세요." 안내 및 남은 시간 카운트다운 | 5회 연속 실패 |
| 세션 만료 | "세션이 만료되었습니다. 다시 로그인해 주세요." 토스트 메시지 | Refresh Token 만료 |

### SCR-002: 메인 대시보드

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-002 |
| 화면명 | 메인 대시보드 |
| URL | /dashboard |
| 설명 | 역할별 맞춤 가젯으로 구성된 프로젝트 현황 대시보드. 가젯별 독립 새로고침 및 레이아웃 커스터마이징 지원 |

#### UI 요소 (역할별 가젯)

| 요소 ID | 유형 | 레이블 | 대상 역할 | 새로고침 주기 | 비고 |
|---------|------|--------|-----------|---------------|------|
| gadget-my-issues | Filter Result | 내 담당 이슈 | Developer | 5분 | 오늘 할 일 |
| gadget-burndown | Chart | Sprint Burndown | Developer, PM | 10분 | 스프린트 잔여량 |
| gadget-velocity | Chart | Velocity Chart | PM, 팀 리더 | 1시간 | 팀 생산성 추이 |
| gadget-sprint-report | Table | Sprint Report | PM, 팀 리더 | 10분 | 완료/미완료 분석 |
| gadget-roadmap | Timeline | Epic 로드맵 | PM, PO | 1시간 | 전체 일정 |
| gadget-pie-type | Pie Chart | 이슈 타입별 분포 | PM, PO | 30분 | 작업 구성 비율 |
| gadget-qa-queue | Filter Result | QA 대기 목록 | QA | 5분 | 테스트 대기 이슈 |
| gadget-created-resolved | Chart | Created vs Resolved | QA | 30분 | 버그 투입/해결 추이 |
| gadget-cfd | Chart | 누적 흐름 다이어그램 | 전체 | 30분 | Cumulative Flow |
| btn-refresh-all | Button | 전체 새로고침 | 전체 | - | 모든 가젯 동시 갱신 |

#### 기능 정의

| 기능 ID | 기능명 | 트리거 | 동작 | API |
|---------|--------|--------|------|-----|
| FN-002-01 | 가젯 로딩 | 페이지 진입 | 역할 기반 가젯 자동 구성 및 데이터 초기 로딩 | GET /dashboard/gadgets |
| FN-002-02 | 이슈 클릭 | 이슈 항목 클릭 | 이슈 상세 화면 이동 | - |
| FN-002-03 | 가젯 커스터마이징 | 설정 버튼 | 가젯 추가/제거/재배치 (드래그앤드롭) | PUT /dashboard/layout |
| FN-002-04 | 가젯 자동 새로고침 | 타이머 만료 | 가젯별 설정 주기에 따라 백그라운드 데이터 갱신 | GET /dashboard/gadgets/{id} |
| FN-002-05 | 가젯 로딩 실패 처리 | API 오류 | 해당 가젯 영역에 "데이터를 불러올 수 없습니다. [새로고침]" 에러 상태 표시. 나머지 가젯 정상 표시 유지 | - |

#### 상태 처리

| 상태 | 표시 내용 | 조건 |
|------|-----------|------|
| 로딩 | 가젯별 스켈레톤 UI 표시 | 초기 데이터 로딩 중 |
| 정상 | 가젯 데이터 표시 | 로딩 완료 |
| 가젯 오류 | 해당 가젯에 에러 메시지 + 수동 새로고침 버튼 표시 | 개별 가젯 API 실패 |
| 전체 오류 | "대시보드를 불러올 수 없습니다." 전체 오류 페이지 | 인증 만료 또는 서버 오류 |

### SCR-003: 백로그

| 항목 | 내용 |

| 화면 ID | SCR-003 |
| 화면명 | 백로그 |

| 요소 ID | 유형 | 레이블 | 필수 | 유효성 검사 | 비고 |

| lst-backlog | List | 백로그 이슈 목록 | - | - | 드래그앤드롭 정렬, 이슈 수 배지 표시 |
| sec-sprint | Section | 스프린트 영역 | - | - | 스프린트별 이슈 그룹, 접기/펼치기 |
| btn-create-issue | Button | 이슈 만들기 | - | - | 인라인 이슈 생성 |
| inp-quick-issue | Input | 이슈 제목 입력 | Y | 최소 2자 | 인라인 생성 시 활성화 |
| sel-epic-filter | Select | 에픽 필터 | - | - | 전체 / 에픽별 필터링 |
| sel-type-filter | Select | 이슈 타입 필터 | - | - | Epic/Story/Task/Bug |
| sel-priority-filter | Select | 우선순위 필터 | - | - | Highest~Lowest |
| sel-assignee-filter | Select | 담당자 필터 | - | - | 팀원 목록 |
| lbl-issue-count | Label | 이슈 개수 | - | - | "백로그 (N개)" 형식 |
| btn-start-sprint | Button | 스프린트 시작 | - | - | 스프린트 영역 헤더 |

#### 기능 정의

| 기능 ID | 기능명 | 트리거 | 동작 | API |
|---------|--------|--------|------|-----|
| FN-003-01 | 백로그 로딩 | 페이지 진입 | 현재 프로젝트의 백로그 이슈 및 스프린트 목록 로딩 | GET /backlog |
| FN-003-02 | 이슈 순서 변경 | 드래그앤드롭 (백로그 내) | 이슈 우선순위 순서 실시간 반영 | PUT /issue/{key} (rank 변경) |
| FN-003-03 | 스프린트 배정 | 이슈를 스프린트 영역으로 드래그 | 해당 이슈를 선택한 스프린트로 이동 | PUT /issue/{key} (sprint 변경) |
| FN-003-04 | 스프린트에서 백로그로 이동 | 이슈를 백로그 영역으로 드래그 | 해당 이슈의 스프린트 배정 해제 | PUT /issue/{key} (sprint = null) |
| FN-003-05 | 인라인 이슈 생성 | btn-create-issue 클릭 | 목록 하단에 인라인 입력 폼 표시, 제목 입력 후 Enter 로 생성 | POST /rest/api/3/issue |
| FN-003-06 | 필터링 | 필터 선택 | 에픽/타입/우선순위/담당자 조건에 맞는 이슈만 표시, 이슈 개수 갱신 | - (클라이언트 필터) |
| FN-003-07 | 이슈 상세 | 이슈 행 클릭 | 이슈 상세 화면(SCR-006)으로 이동 | GET /issue/{key} |

#### 상태 처리

| 상태 | 표시 내용 | 조건 |
|------|-----------|------|
| 빈 백로그 | "백로그에 이슈가 없습니다. 이슈를 생성해 보세요." 빈 상태 안내 | 이슈 0개 |
| 필터 결과 없음 | "조건에 맞는 이슈가 없습니다." | 필터 적용 후 결과 없음 |
| 드래그 중 | 이슈 카드 반투명 처리, 드롭 가능 영역 강조 | 드래그 인터랙션 중 |
| 저장 중 | 이슈 행 로딩 인디케이터 | 순서/스프린트 변경 API 호출 중 |
